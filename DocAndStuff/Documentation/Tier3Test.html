Explanation

 
This program was developed for the testing of s-type instructions.

This program has the symbols:
tier3Test, punch, kick, slap, raichu, entei

This program has the directives:
.start, .data, int.data, str.data, equ, .text, reset.lc, .end

This program has the commands:
lw, lnw, la, adds, subs, lwi, muls, jal, divs, ands, ors, jgt, jlt, j, dump, outn, halt

Errors were included on lines 6, 13, 17, and 23.
