Instruction Table

The Instruction Table contains a list of all valid instructions for the SAL560 language, the type of instruction,
its op code, and its function code. The program will make reference to an imported version of this table in order to
determine whether or not the input program is syntactically corect or not.


This is how the Instruction Table will look in a flat text file ready for inputting.


This is an example of the format the Instruction Table will exist in, in the user guide:


Instruction Table
Instruction Type Op Code Function Code Instruction
I 10 NA addi
S 20 NA jeq
