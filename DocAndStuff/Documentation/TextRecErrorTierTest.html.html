Explanation

The purpose of this test was to ensure that the linker would be able to catch different errors in multiple text records, and to ensure that more than one error could be recorded on one text record.

Results

The anticipated result was that the linker would throw errors for each text record error [errors L-8 through L-15 on the error message list] in the program, and multiple errors on the text record with the incorrect program name field.

The actual result was identical to the expected result.
