Symbol Table

The Symbol Table contains a list all of the variables defined or referenced in the program.
That is to say, it contains the title of the program, all of the labels, the execution start address, etc.
It also conains the location of each symbol, its length, and how it is used. The Symbol Table is sorted by symbol.


This is how the Symbol Table will look in a flat text file after outputting.


This is an example of the format the Symbol Table will exist in:


Symbol Table
Label Location Length Usage Value
A 0 32 Label 0
b 5 32 Label 0
