Explanation

 
This is a test written by Professor Al to aid us in testing our assembler.
It is very simple and tests letter case variety and mobility of the .data section.

This program has the symbols:
altest00, exec.start, OUT, SUM
It also contains DEC1 and DEC which are not defined and should cause an error.

This program uses the directives:
.start, .text, exec.start, nop, .data, int.data, .end

This program uses the commands:
lw, adds, sw, halt

The assembler did not catch the undefined variables.

Test Program

 
.start,altest00,0       | Start Location Counter at zero
.text
    exec.start  0

     ADDs   $1,$1,DEC   | Add second number
     Sw     $1,SUM      | Store SUM
OUT  NOP 
     HALT   0           | Stop execution
.data
SUM  int.data     0 
     Int.data     0 
.end,altest00           |end of Altest00
