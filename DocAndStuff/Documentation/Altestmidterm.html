Explanation

 
Tests a correct program to see if the object file, intermediate file, and symbol table are correctly outputted, as well as the User Report (should show no errors).

Specifically, this test is designed (taken from al's email) to check to see if all operations are properly encoded, and those encodes match up with the real assembled hex/binary values.

This program has the symbols:
Altest_03, Jupiter, Saturn, Pluto, mars, Exec.start


This program uses the directives:
.start, .data, int.data, str.data, .text, ext, exec.start, and .end


This program uses the commands:
lw, mul, addi, divs, srl, sw and halt

This program should not have an error.
