Errata Report

This page contains all currently known errors with our assembler.

As of writing, our linker contains the following errors:

  1. When address location fields for any record is not in proper syntax, the linker will run into a number formatting exception when building the load module.
  2. If a text or linking record is located following an end record, the linker will hit an array index out of bounds exception and abort.

For any of these errors, neither a load module nor a user report will be created.
