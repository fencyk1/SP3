Explanation

 
This program was designed by Professor Al. It is designed to test the capability of the assembler to deal with str.data longer than 1 word.

This program has the symbols:
altest05, xx, yy

This program has the directives:
.start, .data, int.data, str.data, .end

This program has no commands.

The object and intermediate files should have a total of 3 words of data.
Ther should also be an error for invalid integer size.

Test Program

 
.start,altest05,500
.data
xx  int.data 40000000000000
yy  str.data 'ABCDEFGH'    | will this generate 2 words?
.end,altest05

Results

 
While the intermediate file does have the correct amount of data, it is stacked improperly.
The object file does not have the full amount of data.
