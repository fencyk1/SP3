Explanation

 
This test is a modified version of the midterm test, also known as Altest_03. There should not be any errors thrown.
Tests a correct program to see if the object file, intermediate file, and symbol table are correctly outputted, as well as the User Report (should show no errors).

Specifically, this test targets adr.exp to see if the expression is properly encoded.

This program should have the symbols:
Altest_03, exec.start, Jupiter, Saturn, Pluto, Rage, and Mars

This program uses the directives:
.start, .data, int.data, str.data, adr.exp, .text, ext, exec.start, and .end


This program uses the commands:
lw, mul, addi, divs, srl, sw and halt

There was an error at line five involving an opening single quote not being recognized.
