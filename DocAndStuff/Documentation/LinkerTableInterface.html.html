Interface LinkerTableInterface

All Known Implementing Classes: LinkerTable

public interface LinkerTableInterface

This class is an object class. It holds the symbol table from the linker. It has three fields: Name, type, and location, where name is the program or variable name, type is start or ent, and location is the linker assigned memory location.
