Explanation

 
This program was designed to test more of the s-type and i-type instructions.

This program has the symbols:
tier2Test, ff, gg, hh, raichu, exec.start

This program uses the directives:
.start, .data, int.data, str.data, equ, .text, exec.start, .end

This program uses the commands:
lw, lnw, la, lwi, addiu, subiu, muliu, diviu, xori, nori, sw, sa, dump, outn, halt

There were not any errors expected in this program.
