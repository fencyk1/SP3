Explanation

 
This program was designed by Professor Al. It primarily tests for invalid amounts of operands and missing labels.
It also tests index registers.


This program has the symbols:
altest02, exec.start

This program uses the directives:
.start, .data, int.data, .text, reset.lc, exec.start, .end

This program uses the commands:
lw, muls, sw, divs, ands, ors, subs, sll, jal, adds, halt

There are several operand errors in this program, as well as missing labels. They should produce errors at line 9, 15, 18, and 26.

   lw   $0,4
   SlL  $0,$0,2
   sw   $0,15
   JAL  36
   adds $6,$6,0($1)
   halt 200
.end,altest02

Results

 
The assembler spotted the errors at lines 6, 15, 18, and 26.
These were:
6- reset.lc
15- ands
18- ors
26- jal
