Explanation

 
The purpose of this test is to test some of the basic immediate type instructions for correct output, along with the load instructions from the S-types and some IO and the J-type instruction.

This program has the symbols:
tier1Test, aa, bb, cc, dd, ee, exec.start

This program uses the directives:
.start, .data, int.data, str.data, .text, exec.start, .end

This program uses the commands:
lw, lnw, lwi, addi, subi, muli, divi, ori, andi, srv, outc, halt

This test was written expecting not to have errors.
