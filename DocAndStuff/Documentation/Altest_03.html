Explanation

 
Tests a correct program to see if the object file, intermediate file, and symbol table are correctly outputted, as well as the User Report (should show no errors).

This test is designed to test for a general program, that is to say what is expected from a grading session test, and a wide variety of instructions (R/J/S type etc.)

This program should produce the symbols:
YY, ZZ, XX, Err, ERROR, altest03, exec.start

This program uses the directives:
.start, .data, int.data, str.data, .text, ent, exec.start, .end

This program uses the commands:
inn, lw, jeq, lw, mul, muli, div, sw, outn, halt, outc

As expected, this program produced no errors.

     LW      $3,YY

     muli    $5,$4,20    | place intermediate result in reg 5

     outn    1,XX        | output 1 word at location XX
     HALT    0 
ERROR outc   1,Err 
     HALT    250        | Halt and display error code 250
.end,altest03
