Directives Table

The Directives Table contains a list of all legal directives as well as whether or not they impact memory.
So .start would not impact memory, but Hex.data would. The Directives Table is used for determining whether or not
something is a legal/syntactically correct Directive.


This is how the Directives Table will look in a flat text file for importing into the main program.


This is an example of the format the Directievs Table will exist in:


Directives Table
Directive Impacts Memory
.start false
Int.data true
