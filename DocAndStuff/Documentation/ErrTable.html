Error Table

The Error Table contains a list of all of the errors defined and referenced by the parser,
as well as their individual error codes and error messages.


This is how the Error Table will look in a flat text file ready to be imported.


This is an example of the format the Error Table will exist in, in the user guide:

Error Table
Error # Error Name Cause / Error Msg Corrective Action
0 divideByZero "Attempt to divide by zero" Use different value for divisor
1 dataNotHex "The data is not in the correct format" Verify syntax of hex input
