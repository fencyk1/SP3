Explanation

 
This program was designed by Professor Al. It tests the bounds of integers, the nop directive, undefined variables, and addressing.

This program has the variables:
altest04, exec.start, OUT, Sum

This program has two undefined variables:
DEC1, DEC

This program uses the directives:
.start, .text, exec.start, nop, .data, int.data, .end

This program uses the commands:
lw, adds, sw, add, halt

The assembler should catch the undefined variables as errors and also the errors at lines 4, 7, 8, and 9.
